プラットフォーム：スマートフォン

操作	タップ		キャラクターを止めて影を動かせるようになる
	スマホを傾ける　影を動かす

妖精（人参）が右端のゴールへたどり着くとゲームクリアです。妖精は自分で動くので影を動かしてサポートしましょう。

制作　2018年6月頃　2人 プログラムを担当

環境　Unity

スマートフォンでの制作経験を積もうと考え制作しました。ジャイロでの動きを気持ちよくできるようにした工夫をアピールしようと思うので、
申し訳ないのですがスマートフォンで実行していただければ幸いです。
ジャイロでは傾けた方向にすぐに影が動いてしまうと、動かしているという実感がありませんでした。なのでジャイロと画面の結果に
少し遊びを持たせ、また画面の中心を基準に回転するような動きにして動かしている感覚を得られるようにしています。
